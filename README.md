# telegram-teammate-bot

A Telegram bot for finding gaming teammates.

## Features

- **/register**— Create your profile (nickname, game, rank, role)
- **/profile**— View your current profile
- **/lfg**⌄ Post a Looking For Group ad
- **/find**— Browse players filtered by game

## Supported Games

CS2, Valorant, Dota 2, Fortnite, Other

## Setup

1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Set your `BOT_TOKEN` environment variable
4. Run: `python bot.py`
